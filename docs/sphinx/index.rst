
sdss_brain's documentation
=============================================

This is the documentation for the SDSS Python product sdss_brain. The current version is |sdss_brain_version|. You can install the package by doing

.. code-block:: console

  $ pip install sdss-brain

Or to install with extras

.. code-block:: console

  $ pip install sdss-brain[extras]
